# 数据库设计说明

> 本系统使用 MySQL 8.0 数据库，包含 17 张 InnoDB 表，对应论文 4.3 节物理表设计。

| 编号 | 表名 | 中文名 | 主要用途 |
|------|------|-------|---------|
| 1 | sys_user | 系统用户表 | 账号、密码、角色、状态 |
| 2 | sys_role | 系统角色表 | 角色定义 |

| 6 | bridge | 桥梁基础参数表 | α 气候系数、AADT |
| 7 | bridge_aadt_history | AADT 历史表 | 交通量变更记录 |
| 8 | monitoring_record | 监测推送表 | IoT 数据 |

| 10 | device_category | 设备分类表 | 11 类设备字典 |

| 字段 | 类型 | 说明 |

| alpha_coef | DECIMAL(4,2) | α 气候系数 |
| current_aadt | INT | 当前 AADT |
| beta_coef | DECIMAL(4,2) | β AADT 系数 |

**α 气候系数取值：**
| 气候带 | 代码 | α 值 |
|-------|------|------|
| 寒冷地区 | cold | 1.30 |
| 严寒地区 | severe_cold | 1.50 |
| 温和地区 | temperate | 1.00 |
| 湿热地区 | humid | 1.10 |
| 沿海地区 | coastal | 1.20 |

**β AADT 系数阶梯：**
| AADT 范围 | β 值 |
|---------|------|
| < 5000 | 1.00 |
| 5000 - 10000 | 1.10 |
| 10000 - 20000 | 1.20 |
| 20000 - 30000 | 1.30 |
| ≥ 30000 | 1.50 |

## 🌱 种子数据规模

数据库初始化脚本包含：

- **8 个用户账号**：admin、engineer、purchaser、maintainer、scrap 等
- **5 种角色**：系统管理员、工程管理员、采购员、运维员、报废鉴定员
- **5 座桥梁**：赣江大桥、鄱阳湖大桥、东郭虎渠桥、雪山高架桥、松花江北桥
- **11 类设备分类**：拉索、伸缩缝、支座、桥面铺装、防撞栏、照明系统、监测传感器等
- **38 台设备**：分布在不同桥梁，状态多样化
- **历史数据**：保养记录、故障工单、采购订单等

## 🔧 核心业务规则

### TCO 决策算法

```
维修方案 TCO = 修复费 + 未来保养费 × N年 + 未来维修费 × N年
更换方案 TCO = 新设备价 + 未来保养费 × N年 - 残值
```

### 残值估算公式

```
残值 = 购置价 × (1 - 已用年限 / 设计寿命) × 市场折扣系数(0.70)
```

### α×β 动态保养周期

```
实际保养周期 = 标准保养周期 × α × β
```

## 📊 8 项关键率计算

| 指标 | 计算公式 |
|------|---------|
| 设备在用率 | 使用中设备数 / 总设备数 × 100% |
| 设备完好率 | 非故障非报废设备数 / 总设备数 × 100% |
| 年故障率 | 年故障工单数 / 总设备数 × 100% |
| 保养完成率 | 已完成保养数 / 应保养数 × 100% |
| 工单关闭率 | 已关闭工单数 / 总工单数 × 100% |
| MTBF | 总运行时间 / 故障次数（小时） |
| MTTR | 总修复时间 / 工单数（小时） |
| 累计运维成本 | Σ保养费 + Σ维修费 |

## 🔄 数据库管理

### 重置数据库

```bash
mysql -u root -p -e "DROP DATABASE IF EXISTS bridge_lifecycle;"
mysql -u root -p -e "CREATE DATABASE bridge_lifecycle CHARACTER SET utf8mb4;"
mysql -u root -p bridge_lifecycle < backend/src/main/resources/schema.sql
mysql -u root -p bridge_lifecycle < backend/src/main/resources/data.sql
```

```sql
USE bridge_lifecycle;
```
